### How to play (Control)

-   Mouse Operation : Ship Moving
-   Space key : Operates & Shooting
-   Special mode: If you don't shoot for a while.

### Requirement

-   TypeScript, JavaScript

### How to Build & Run

```sh
$ cd barrage
$ ./node_module/.bin/tsc
$ npm start
```

Browse http://127.9.9.1:8080
